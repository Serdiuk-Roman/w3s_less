Input Restrictions

Attribute Description
checked Вказує, що поле введення має бути попередньо вибрано під час завантаження сторінки (для type="checkbox" або type="radio")
disabled Вказує, що поле введення має бути вимкнене
max Визначає максимальне значення для поля введення
maxlength Визначає максимальну кількість символів для поля введення
min Визначає мінімальне значення для поля введення
pattern Визначає регулярний вираз для перевірки вхідного значення
readonly Вказує, що поле введення лише для читання (неможливо змінити)
required Вказує, що поле введення обов’язкове (має бути заповнене)
size Визначає ширину (в символах) поля введення
step Визначає допустимі інтервали чисел для поля введення
value Визначає значення за умовчанням для поля введення
