HTML Style Guide

  • Декларуй тип документа(версію html)
  • Назви елементів, тегів і атрибутів у нижньому регістрі
  • Значення атрибутів у лапках
  • Закриті всі елементи навіть одинарні
  • Для зображень використовуй значення ширини, висоти та альтернативний опис.
  • Уникай довгих рядків коду
  • В міру використовуй відступи та пусті рядки
  • Не пропускай теги html, head, title та body
  • Добавляй атрибут мови, кодування та viewport
  • Залишай коментарі
