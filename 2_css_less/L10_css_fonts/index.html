Font Family

У CSS існує п’ять загальних сімейств шрифтів:

  1. Serif шрифти мають невелику риску по краях кожної літери. Вони створюють відчуття офіційності та елегантності.
  2. Sans-serif шрифти мають чіткі лінії (без дрібних штрихів). Вони створюють сучасний і мінімалістичний вигляд.
  3. Monospace шрифти - тут усі літери мають однакову фіксовану ширину. Вони створюють механічний вигляд.
  4. Cursive шрифти імітують людський почерк.
  5. Fantasy — це декоративні/грайливі шрифти.

Web Safe Fonts

Нижче наведено список найкращих веб-безпечних шрифтів для HTML і CSS:

  • Arial (sans-serif)
  • Verdana (sans-serif)
  • Tahoma (sans-serif)
  • Trebuchet MS (sans-serif)
  • Times New Roman (serif)
  • Georgia (serif)
  • Garamond (serif)
  • Courier New (monospace)
  • Brush Script MT (cursive)

Example:

  • This is a Serif Fonts
    font-family: "Times New Roman", Times, serif;
  • This is a Sans-Serif Fonts
    font-family: Tahoma, Verdana, sans-serif;
  • This is a Monospace Fonts
    font-family: "Courier New", Courier, monospace;
  • This is a Cursive Fonts
    font-family: "Brush Script MT", cursive;
  • This is a Fantasy Fonts
    font-family: Copperplate, Papyrus, fantasy;

Font Style

Властивість font-style здебільшого використовується для визначення тексту курсивом.

  • normal - Текст відображається нормально
  • italic - Текст виділяється курсивом
  • oblique - Текст "нахилений" (косий дуже схожий на курсив, але менш підтримується)

Властивість font-weight визначає щільність шрифту:

  • lighter
  • normal
  • bold
  • bolder

Властивість font-variant визначає, чи повинен текст відображатися шрифтом із зменшеними великими літерами.

Font Size

Властивість font-size встановлює розмір тексту.

Примітка. Якщо ви не вкажете розмір шрифту, розмір за замовчуванням для звичайного тексту, наприклад абзаців, становить 16 пікселів (16 пікселів=1 ем). 1em дорівнює поточному розміру шрифту. Стандартний розмір тексту в браузерах становить 16 пікселів.

Great Font Pairings (Поєднання шрифтів)

  1. Доповнювати (Чудове поєднання шрифтів має гармоніювати, не бути надто схожими чи надто різними)
  2. Використовуйте суперсімейства шрифтів (Lucida Sans, Lucida Serif, Lucida Typewriter Sans, Lucida Typewriter Serif і Lucida Math.)
  3. Контраст - король (Комбінування без засічок із засічками є добре відомим поєднанням)
  4. Виберіть лише одного боса (зміна розміру, ваги та коліру)

Властивість шрифту

  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family

Приклад: font: italic bold 12px/30px Georgia, serif;

Примітка: значення font-size і font-family є обов’язковими.
