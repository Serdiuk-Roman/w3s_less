CSS Margins & Padding

Margins(поля) використовуються для створення простору навколо елементів, зовні будь-яких визначених рамок.

Padding(відступ) використовується для створення простору навколо вмісту елемента всередині будь-яких визначених меж(borders).

Margin Значення

  • auto - браузер обчислює поля
  • length - sвказує поля в px, pt, cm, і т. ін.
  • % - визначає відступ у % від ширини елемента, що містить
  • inherit - вказує, що поля мають бути успадковані від батьківського елемента
  • Від’ємні значення дозволені

margin: auto; - Вирівняти блок по центру

Margin Shorthand

  • margin: 25px 50px 75px;
    • top margin is 25px
    • right and left margins are 50px
    • bottom margin is 75px

Margin Collapse

Згортання полів(margin collapse) відбувається, коли вертикально суміжні поля елементів блоків стикаються, щоб розділити загальний простір полів. Розмір цього спільного простору визначається більшим числовим значенням поля(margin) блоків.
