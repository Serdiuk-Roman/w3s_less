CSS Links

За допомогою CSS, посилання можна стилізувати різними способами.

Styling

Посилання можна стилізувати за допомогою будь-якої властивості CSS (наприклад: color, font-family, background тощо).

Чотири стани посилань:

  • a:link - звичайне, невідвідуване посилання
  • a:visited - посилання, яке відвідав користувач
  • a:hover - посилання, коли користувач наводить на нього мишку
  • a:active - посилання в момент натискання

a:hover ПОВИНЕН бути після a:link і a:visited
a:active ПОВИНЕН бути після a:hover

Text Decoration

Властивість text-decoration здебільшого використовується для видалення підкреслень із посилань: 

        a:link {
          text-decoration: none;
          background-color: yellow;
        }
