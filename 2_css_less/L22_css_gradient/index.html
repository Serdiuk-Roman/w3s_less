Gradients

Градієнти CSS дозволяють відображати плавні переходи між двома або більше вказаними кольорами.

CSS визначено три типи градієнтів:

  • Лінійні градієнти
  • Радіальні градієнти (від центру)
  • Конічні градієнти (обертаються навколо центральної точки)

Linear Gradients

Необхідно визначити принаймні два переходи кольорову. Кольорові позначки – це кольори, між якими ви хочете зробити плавні переходи. Ви також можете встановити початкову точку та напрямок (або кут) разом із ефектом градієнта.

background-image: linear-gradient(to bottom right, lightblue , yellow);

У цьому прикладі показано лінійний градієнт із декількома кольорами

background-image: linear-gradient(60deg, red,orange,yellow,green,blue,indigo,violet);

Radial Gradients

Радіальний градієнт визначається його центром. Щоб створити радіальний градієнт, ви також повинні визначити принаймні дві кольорові точки.

background-image: radial-gradient(red, yellow, green);

Conic Gradients

Конічний градієнт — це градієнт із переходами кольорів, обертаними навколо центральної точки. Щоб створити конічний градієнт, необхідно визначити принаймні два кольори.
