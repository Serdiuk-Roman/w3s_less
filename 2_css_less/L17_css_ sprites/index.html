Image Sprites

Спрайт зображення — це набір зображень, складених в одне зображення.

Використання спрайтів зображень зменшить кількість запитів до сервера та заощадить пропускну здатність.

Simple Example

Замість використання трьох окремих зображень ми використовуємо одне зображення

За допомогою CSS ми можемо показати лише ту частину зображення, яка нам потрібна. 

          #prev {
            width: 43px;
            height: 44px;
            background: url(img_navsprites.gif) -47px -45px;
          }
          #prev:hover {
            background: url("../img/img_navsprites.gif") -47px 0;
          }
