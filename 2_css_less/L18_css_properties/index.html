CSS Counters

Лічильники CSS схожі на "змінні". Значення змінних можна збільшити за допомогою правил CSS (які відстежуватимуть, скільки разів вони використовуються).

  • counter-reset - Створює або скидає лічильник
  • counter-increment - Збільшує значення лічильника
  • content - Вставляє створений вміст
  • counter() або counters() функція - додає значення лічильника до елемента

CSS Units

CSS має кілька різних одиниць для вираження довжини.

Розрізняють два типи одиниць вимірювання довжини: абсолютні та відносні.

Абсолютні довжини

Абсолютні одиниці довжини є фіксованими, і довжина, виражена в будь-якій з них, відображатиметься як саме той розмір.

Unit Description
cm centimeters
mm millimeters
in inches (1in = 96px = 2.54cm)
px pixels (1px = 1/96th of 1in)
pt points (1pt = 1/72 of 1in)
pc picas (1pc = 12 pt)

Відносні довжини

Одиниці відносної довжини визначають довжину відносно іншої властивості довжини. Одиниці відносної довжини краще масштабуються між різними середовищами візуалізації.

Unit Description
em Відносно розміру шрифту елемента (2em означає 2-кратний розмір поточного шрифту)
ex Відносно висоти символа "х" поточного шрифту (використовується рідко)
ch Відносно ширини "0" (нуль)
rem Відносно розміру шрифту кореневого елемента
vw Відносно 1% ширини вікна перегляду
vh Відносно 1% висоти вікна перегляду
vmin Відносно 1% меншого розміру вікна перегляду
vmax Відносно 1% більшого розміру вікна перегляду
% Відносно батьківського елемента

Порада: одиниці em і rem практичні у створенні ідеально масштабованого макета! * Viewport = розмір вікна браузера. Якщо вікно перегляду має ширину 50 см, 1vw = 0,5 см.

The !important Rule

Правило !important у CSS використовується для того, щоб додати властивості/значенню більшої важливості, ніж зазвичай.

Math Functions

Математичні функції CSS дозволяють використовувати математичні вирази як значення властивостей. Тут ми пояснимо функції calc(), max() і min().

Function Description
calc() Дозволяє виконувати обчислення для визначення значень властивостей CSS. Можна використовувати такі оператори: + - * /
max() Використовує найбільше значення зі списку значень, розділених комами, як значення властивості
min() Використовує найменше значення зі списку значень, розділених комами, як значення властивості
