CSS Colors

Кольори вказуються за допомогою попередньо визначених назв кольорів або значень RGB, HEX, HSL, RGBA, HSLA.

Назви

CSS/HTML підтримує 140 стандартних назв кольорів.

Tomato DodgerBlue Blue MediumSeaGreen Violet Orange SlateBlue Gray LightGray


RGB

rgb(red, green, blue):

rgb(255, 0, 0) rgb(0, 0, 255) rgb(60, 179, 113) rgb(238, 130, 238) rgb(255, 165, 0) rgb(106, 90, 205)


RGBA

rgba(red, green, blue, alpha):

rgba(255, 99, 71, 0) rgba(255, 99, 71, 0.2) rgba(255, 99, 71, 0.4) rgba(255, 99, 71, 0.6) rgba(255, 99, 71, 0.8) rgba(255, 99, 71, 1)


HEX

#rrggbb
Де rr (червоний), gg (зелений) і bb (синій) є шістнадцятковими значеннями від 00 до ff (так само, як десяткові 0-255).

#ff0000 #0000ff #3cb371 #ee82ee #ffa500 #6a5acd


Короткий запис: #ff00cc ->#f0c

HSL

Колір можна вказати за допомогою відтінку(hue), насиченості(saturation) та освітленості(lightness) у формі:

hsl(hue, saturation, lightness)

hsl(0, 100%, 50%) hsl(240, 100%, 50%) hsl(147, 50%, 47%) hsl(300, 76%, 72%) hsl(39, 100%, 50%) hsl(248, 53%, 58%)


HSLA

hsla(hue, saturation, lightness, alpha)

hsla(9, 100%, 64%, 0) hsla(9, 100%, 64%, 0.2) hsla(9, 100%, 64%, 0.4) hsla(9, 100%, 64%, 0.6) hsla(9, 100%, 64%, 0.8) hsla(9, 100%, 64%, 1)
