Float and clear

Властивість CSS float визначає, як елемент має плавати.

Властивість CSS clear визначає, які елементи можуть плавати поруч із очищеним елементом і з якого боку.

Float

Властивість float використовується для розміщення та форматування вмісту, наприклад. нехай зображення плаває ліворуч до тексту в контейнері.

  • left - Елемент плаває з лівої сторони свого контейнера
  • right - Елемент плаває з правої сторони свого контейнера
  • none - Елемент не плаває (відображатиметься лише там, де він зустрічається в тексті). Це за умовчанням
  • inherit - Елемент успадковує значення float своїх батьків

Clear

Коли ми використовуємо властивість float і хочемо, щоб наступний елемент був нижче (не праворуч чи ліворуч), нам доведеться використовувати властивість clear.

  • none - Елемент не проштовхується нижче лівих або правих плаваючих елементів. Це за умовчанням
  • left - Елемент виштовхується нижче лівих плаваючих елементів
  • right - Елемент виштовхується нижче правих плаваючих елементів
  • both - Елемент виштовхується нижче лівого та правого плаваючих елементів
  • inherit - Елемент успадковує значення clear від свого батька
