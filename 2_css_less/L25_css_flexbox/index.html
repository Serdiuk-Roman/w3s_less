Властивості контейнера CSS Flexbox

Property Description
align-content Змінює поведінку властивості flex-wrap. Він схожий на align-items, але замість вирівнювання гнучких елементів він вирівнює гнучкі лінії
align-items Вертикально вирівнює гнучкі елементи, якщо вони не займають весь доступний простір на поперечній осі
display Визначає тип поля, що використовується для елемента HTML
flex-direction Визначає напрямок гнучких елементів усередині гнучкого контейнера
flex-flow Скорочена властивість для flex-direction і flex-wrap
flex-wrap Визначає, чи мають обгортатися елементи flex чи ні, якщо для них недостатньо місця на одній лінії flex
justify-content Горизонтально вирівнює гнучкі елементи, якщо вони не займають весь доступний простір на головній осі

Flex Items

Прямі дочірні елементи flex контейнера автоматично стають гнучкими (гнучкими) елементами.

Властивості гнучкого елемента

Property Description
align-self Визначає вирівнювання для елемента flex (перевизначає властивість align-items контейнера flex)
flex Скорочена властивість властивостей flex-grow, flex-shrink і flex-basis
flex-basis Визначає початкову довжину елемента flex
flex-grow Визначає, наскільки збільшуватиметься flex-елемент відносно решти flex-елементів у тому самому контейнері
flex-shrink Визначає, наскільки flex-елемент зменшиться відносно решти flex-елементів у тому самому контейнері
order Визначає порядок елементів Flex в одному контейнері

order

Властивість order визначає порядок гнучких елементів.

flex-grow

Властивість flex-grow визначає, наскільки flex-елемент буде рости відносно решти flex-елементів.

flex-shrink

Властивість flex-shrink визначає, наскільки flex-елемент зменшиться відносно решти flex-елементів.

flex-basis

Властивість flex-basis визначає початкову довжину елемента flex.

flex

Властивість flex — це скорочення властивостей flex-grow, flex-shrink і flex-basis.

align-self

Властивість align-self визначає вирівнювання для вибраного елемента всередині гнучкого контейнера.

Flex Responsive (адаптивний, чутливий)

За допомогою @media ви можете використовувати медіа-запити для створення різних макетів для різних розмірів екрана та пристроїв. 

      /* Адаптивний макет – якщо ширина екрана менше 700 пікселів
      то дві колонки складаються одна над одною,
      інакше поруч один з одним */
    .navbar {
      display: flex;
    }
    .row {
      display: flex;
      flex-wrap: wrap;
    }
    @media screen and (max-width: 700px) {
      .row, .navbar {
        flex-direction: column;
      }
    }
