CSS Display

Властивість display є найважливішою властивістю CSS для керування макетом.

За замовчуванням відображення для більшості елементів є блоковим (block) або рядковим (inline).

display: none; зазвичай використовується з JavaScript, щоб приховати та показати елементи без їх видалення та повторного створення.

visibility:hidden; - приховує елемент. Однак елемент займатиме стільки ж місця, скільки й раніше. Елемент буде приховано, але все одно вплине на макет.

The position Property

Властивість position вказує тип методу позиціонування, що використовується для елемента:

  • статичний (static)
  • відносний (relative)
  • фіксований (fixed)
  • абсолютний (absolute)
  • закріплений (sticky)

position: static;

За замовчуванням в браузерах. На статично позиціоновані елементи не впливають властивості top, bottom, left і right.

position: relative; left: 30px;

Елемент із позицією: relative; розташований відносно свого нормального положення:

position: fixed;
Зверніть увагу на фіксований елемент у нижньому правому куті сторінки.

position: fixed;

Елемент із положенням: фіксоване; розташований відносно області перегляду, що означає, що він завжди залишається в тому самому місці, навіть якщо сторінку прокручують.

The z-index Property

Властивість z-index визначає порядок стека елемента.

Елементи можуть перекривати інші елементи. Властивість z-index визначає який елемент має бути розміщений попереду чи позаду інших.

Для прикладу

Контейнер цього абзацу з зображенням має position: relative;,
а зображення розміщено absolute (абсолютно) і має негативний індекс z-index: -1: , тому картинка розташувалась під текстом.

Примітка: z-index працює лише з позиціонованими елементами (position: absolute, position: relative, position: fixed, або position: sticky) і гнучкими елементами(flex items) - це елементами, які є прямими дочірніми елементами display: flex елементів.

Overflow

Властивість overflow (перелив) CSS контролює, що відбувається з вмістом, який занадто великий, щоб поміститися в область.
