Transitions

Переходи CSS дозволяють плавно змінювати значення властивостей протягом певного часу.

  • transition
  • transition-delay
  • transition-duration
  • transition-property
  • transition-timing-function
Наведіть курсор на елемент, щоб побачити ефект

transition: property duration timing-function delay;

Властивість transition-timing-function визначає криву швидкості ефекту переходу.

  • ease - визначає ефект переходу з повільним початком, потім швидким і повільним завершенням (це типово)
  • linear - визначає ефект переходу з однаковою швидкістю від початку до кінця
  • ease-in - визначає ефект переходу з повільним стартом
  • ease-out - визначає ефект переходу з повільним кінцем
  • ease-in-out - визначає ефект переходу з повільним початком і закінченням
  • cubic-bezier(n,n,n,n) - дозволяє визначати власні значення у функції кривих Біз’є

Animations

CSS дозволяє анімувати елементи HTML без використання JavaScript або Flash!

  • @keyframes
  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode
  • animation

Щоб використовувати CSS-анімацію, ви повинні спочатку вказати кілька ключових кадрів для анімації. Ключові кадри містять стилі, які матиме елемент у певний час. 

  @keyframes example {
    0%   {background-color:red; left:0px; top:0px;}
    25%  {background-color:yellow; left:200px; top:0px;}
    50%  {background-color:blue; left:200px; top:200px;}
    75%  {background-color:green; left:0px; top:200px;}
    100% {background-color:red; left:0px; top:0px;}
  }

Властивість animation-timing-function визначає криву швидкості анімації.

linear
ease
ease-in
ease-out
ease-in-out

Скорочений запис анімації

    div {
      animation-name: example;
      animation-duration: 5s;
      animation-timing-function: linear;
      animation-delay: 2s;
      animation-iteration-count: infinite;
      animation-direction: alternate;
    }

==

    div {
      animation: example 5s linear 2s infinite alternate;
    }
