Адаптивна таблиця

Адаптивна таблиця відображатиме горизонтальну смугу прокрутки, якщо екран замалий для відображення всього вмісту.

Додамо елемент-контейнер (наприклад, <div>) з overflow-x:auto навколо елемента <table>, щоб зробити його адаптивним.
