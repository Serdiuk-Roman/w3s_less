Navigation Bar

Для будь-якого веб-сайту важлива проста у використанні навігація.

За допомогою CSS ви можете перетворити нудні HTML-меню на гарні навігаційні панелі.

Navigation Bar = List of Links

Панель навігації — це в основному список посилань, тому використання елементів <ul> і <li> має сенс

  • Home
  • News
  • Contact
  • About

Примітка: ми використовуємо href="#" для тестових посилань. На реальному веб-сайті це були б URL-адреси.
