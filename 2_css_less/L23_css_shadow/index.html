Shadow Effects

За допомогою CSS ви можете додавати тінь до тексту та елементів. У цих розділах ви дізнаєтеся про такі властивості:

  • text-shadow
  • box-shadow

Text Shadow

Властивість CSS text-shadow додає тінь до тексту.

        h2.t-sh {
          color: white;
          text-shadow:  1px 1px 2px black,
                        0 0 25px red,
                        0 0 5px darkblue;
        }

Box Shadow

          .b-sh {
            width: 480px;
            height: 240px;
            padding: 16px;
            background-color: coral;
            border-radius: 16px;
            box-shadow: 10px 10px 5px 12px blue,
                        16px 16px 5px lightgreen inset;
          }
