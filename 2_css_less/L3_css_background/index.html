CSS Backgrounds

Властивості фону CSS використовуються для додавання фонових ефектів для елементів.

Color

background-color: red;

  • дійсна назва кольору, наприклад "red"
  • шістнадцяткове значення, наприклад "#ff0000"
  • значення RGB, наприклад "rgb(255,0,0)"

opacity:0.5;

background: rgba(255, 0, 0, 0.5);


Image

Image as the background!

background-repeat: no-repeat;
background-position: right top;

Властивість background-position використовується
для визначення положення фонового зображення.


Background Attachment

background-repeat: no-repeat;
background-position: center top;
background-attachment: fixed;

Властивість background-attachment визначає,
чи має фонове зображення прокручуватися чи
бути фіксованим (не прокручуватиметься
разом із рештою сторінки):

background-repeat: no-repeat;
background-position: center top;
background-attachment: scroll;

бути фіксованим (не прокручуватиметься
разом із рештою сторінки):

Background Shorthand

Короткий запис

background-color: #ffffff;
background-image: url("img_tree.png");
background-repeat: no-repeat;
background-position: right top;

Еквівалентно

background: #ffffff url("img_tree.png") no-repeat right top;


Інші

background-clip - Визначає область малювання фону
background-origin - Вказує, де розташовано фонове зображення
background-size - Вказує розмір фонового(их) зображення(ів)
