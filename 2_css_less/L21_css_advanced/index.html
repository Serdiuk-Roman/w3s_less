The object-fit Property

Властивість CSS object-fit використовується для визначення того, як <img> або <video> слід змінювати розмір, щоб відповідати його контейнеру.

  • fill - Це за умовчанням. Зображення змінюється, щоб заповнити заданий розмір. За потреби зображення буде розтягнуто або зім’ято, щоб підходити
  • contain - Зображення зберігає співвідношення сторін, але його розмір змінюється відповідно до вказаного розміру
  • cover - Зображення зберігає своє співвідношення сторін і заповнює заданий розмір. Зображення буде обрізано відповідно до розміру
  • none - Розмір зображення не змінюється
  • scale-down - зображення зменшено до найменшої версії none або contain

The object-position Property

Властивість CSS object-position використовується для визначення того, як <img> або <video> має розташовуватися в контейнері. 

        img {
          width: 200px;
          height: 300px;
          object-fit: cover;
          object-position: 15% 100%;
        }

Resizing

Натисніть і перетягніть нижній правий кут цього елемента.

У багатьох браузерах textarea змінюється за замовчуванням. Можна використати властивість resize, щоб вимкнути зміну розміру.

Outline Offset

Властивість outline-offset додає проміжок між outline (контуром) і краєм або межею елемента. Контур НЕ є частиною розмірів елемента; ширина контуру не впливає на загальну ширину та висоту елемента. Контур малюється за межами елемента та може перекривати інший вміст.

Box Sizing

Властивість CSS box-sizing дозволяє нам включити відступ і рамку в загальну ширину і висоту елемента.

Value Description
content-box За замовчуванням. Властивості ширини та висоти (і властивості min/max) включають лише вміст. Рамки та відступи не включені
border-box Властивості ширини та висоти (і властивості min/max) включають вміст, відступи та межі
initial Встановлює для цієї властивості значення за умовчанням.
inherit Успадковує цю властивість від батьківського елемента.

Media Queries

Media Query Syntax

        @media not|only mediatype and (expressions) {
          CSS-Code;
        }

Media Types

Value Description
all Використовується для пристроїв усіх типів носіїв
print Використовується для принтерів
screen Використовується для екранів комп’ютерів, планшетів, смартфонів тощо.
speech Використовується для програм зчитування з екрана, які "читають" сторінку вголос

Example

        @media screen and (min-width: 480px) {
          #leftsidebar {width: 200px; float: left;}
          #main {margin-left: 216px;}
        }
