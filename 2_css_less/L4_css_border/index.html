CSS Borders

Властивості CSS border дозволяють вказати стиль, ширину та колір рамки елемента.

Border Style

Властивість border-style визначає тип рамки для відображення.

  • dotted - пунктир
  • dashed - штрих
  • solid - суцільна
  • double - подвоєна
  • groove - рамка з 3D єфектом
  • ridge - рамка з 3D єфектом
  • inset - рамка з 3D єфектом
  • outset - рамка з 3D єфектом
  • none - рамка відсутння
  • hidden - прихована
  • Стилі можна комбінувати ___↑____→___↓____←__
    ____________border-style: dotted dashed solid double;

Border Width

border-width: 5px;
px, pt, cm, em або за допомогою одного з трьох
попередньо визначених значень: thin, medium, or thick.

Border Color

  • name - like "red"
  • HEX - "#ff0000"
  • RGB - "rgb(255,0,0)"
  • HSL - "hsl(0, 100%, 50%)"
  • transparent (прозорий)

Individual Sides

border-style: dotted solid double dashed;

  • top border is dotted
  • right border is solid
  • bottom border is double
  • left border is dashed

Shorthand

border-left: 48px solid red;

Rounded Borders

border: 2px solid red;
border-radius: 8px;
