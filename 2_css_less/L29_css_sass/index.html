Sass

Sass Introduction

Sass дозволяє використовувати фішки, яких немає в CSS, як-от змінні, вкладені правила, міксини, імпорт, успадкування, вбудовані функції та інші речі.

  • Sass не залежить від платформи, операційної системи
  • Sass працює в Edge/IE (починаючи з IE 8), Firefox, Chrome, Safari, Opera
  • Sass базується на Ruby (мові програмування)

Variables

Змінні — це спосіб зберігати інформацію, яку можна використовувати пізніше. За допомогою Sass ви можете зберігати інформацію у змінних, наприклад:

  • strings
  • numbers
  • colors
  • booleans
  • lists
  • nulls
        $myFont: Helvetica, sans-serif;
        $myColor: red;

        body {
          font-family: $myFont;
          color: $myColor;
        }

Importing

Ви можете створювати невеликі файли з фрагментами CSS для включення в інші файли Sass. Прикладами таких файлів можуть бути: файл скидання, змінні, кольори, шрифти, розміри шрифтів тощо. 

@import filename;

mixin

        @mixin mixin-name {
          property: value;
          property: value;
          ...
        }
        selector {
          @include mixin-name;
        }

extend

      .button-basic  {
        border: none;
        padding: 15px 30px;
        text-align: center;
        font-size: 16px;
        cursor: pointer;
      }
      .button-submit  {
        @extend .button-basic;
        background-color: green;
        color: white;
      }
